# Installation

## Gradle (Groovy)

```groovy
implementation 'com.skyecodes:karith:0.2.0'
```

## Gradle (Kotlin)

```kotlin
implementation("com.skyecodes:karith:0.2.0")
```

## Maven

```xml
<dependency>
    <groupId>com.skyecodes</groupId>
    <artifactId>karith</artifactId>
    <version>0.2.0</version>
</dependency>
```
